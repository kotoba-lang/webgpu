# Rust Fixture Inventory

Status: no Rust fixtures remain in this repository.

`kami-webgpu` is a CLJ/CLJS/CLJC repository with no `Cargo.toml`, no Cargo
workspace, and no tracked `.rs` files.

The former generated fixture:

- `fixtures/pipeline_specs.rs`

has been replaced with:

- `fixtures/pipeline_specs.edn`

`fixtures/pipeline_specs.edn` is generated by `bb gen-pipeline-specs` from
`kami.pipelines/native-pipelines`. This keeps the render-pipeline contract as
EDN data and avoids keeping native-shaped Rust output in this repo.

Policy:

- Do not add a `Cargo.toml` to this repo.
- Do not commit generated `.rs` fixtures.
- Native adapters should consume CLJC/EDN contracts from their own repositories.
